# Mommy's Soles - E621 Android Client

Welcome to the **Mommy's Soles** repository. This project is a feature-rich, unofficial Android client designed for the e621.net and e926.net image boards, providing a seamless and native mobile experience for browsing and interacting with content.

This repository contains:
- **Source Code**: The complete Android Studio project source code is located in the `E621Client/` directory.
- **Release APK**: The latest compiled application package (`mommy-s-soles-v1.0.0.apk`) is available in the root directory for quick installation.
- **Assets**: Project assets and resources.

## ‚ú® Key Features

### Browsing & Search
- **Infinite Scrolling**: Effortlessly browse through thousands of posts with smooth infinite scroll pagination.
- **Advanced Search**: Powerful tag-based search engine with intelligent autocomplete suggestions to help you find exactly what you're looking for.
- **Media Support**: Full support for all e621 media types, including high-resolution images, animated GIFs, and WebM videos.
- **Immersive Viewer**: Full-screen media viewer with pinch-to-zoom and gesture controls.

### Account Integration
- **Secure Login**: Authenticate securely using your e621 API key.
- **Favorites Management**: Easily view, add, and remove favorites.
- **Voting System**: Upvote or downvote posts directly from the app.
- **User History**: Access your uploaded posts, comments, and interaction history.
- **DMail**: Read and send Direct Messages.
- **Sets**: Create and manage post sets.

### Installation
To install the application on your Android device:
1. Download the `mommy-s-soles-v1.0.0.apk` file from this repository.
2. Enable "Install from Unknown Sources" in your device settings if prompted.
3. Install the APK and launch the app.

### Building from Source
To build the project yourself:
1. Navigate to the `E621Client/` directory.
2. Open the project in Android Studio.
3. Sync Gradle project files.
4. Build and Run on your emulator or connected device.
